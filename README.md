# 太空捕手 - 豪華進化版 (Space Catcher Deluxe)

這是一個使用 Pygame 開發的休閒閃避接物遊戲。玩家需要操控飛行員捕捉金幣，同時避開危險的紅色假金幣。遊戲包含商店升級系統、多位角色以及多種特殊道具。

## 遊戲特色

*   **無盡挑戰**：隨著分數增加，遊戲速度會逐漸加快，挑戰你的反應極限。
*   **多樣化道具**：
    *   **金幣 (Gold Coin)**：主要得分來源。
    *   **紅色假金幣 (Fake Coin)**：危險障礙物，碰到會消耗護盾或導致遊戲結束。
    *   **星星 (Star)**：獲得短暫無敵狀態，可以直接撞毀假金幣。
    *   **火焰花 (Fire Flower)**：開啟火焰模式，自動發射火球清除障礙物。
*   **商店系統**：使用收集到的代幣升級你的裝備：
    *   **磁鐵**：增加金幣吸引範圍。
    *   **護盾**：增加可抵擋傷害的次數。
    *   **動力**：提升角色的基礎移動速度。
*   **角色選擇**：提供兩位不同的飛行員可供選擇（透過更換圖片或預設圖形）。
*   **狀態機架構**：遊戲代碼採用狀態機設計（主選單、商店、選角、遊戲中），結構清晰。

## 系統需求

*   Python 3.x
*   Pygame

## 安裝與執行

1.  確保已安裝 Python。
2.  安裝 Pygame 套件：
    ```bash
    pip install pygame
    ```
3.  執行遊戲：
    ```bash
    python 嘉桐gemini.py
    ```

## 遊戲操作說明

### 選單介面
*   **滑鼠點擊**：選擇選項 (開始遊戲、商店、結束)。
*   **鍵盤對應**：
    *   `1`: 進入選角畫面
    *   `2`: 進入商店
    *   `3`: 結束遊戲

### 遊戲中
*   **左右方向鍵 (`←`, `→`)**：控制角色移動。
*   **空白鍵 (`Space`)**：跳躍 / 遊戲結束後重玩。
*   **`P`**：暫停 / 繼續遊戲。
*   **`S`**：開啟商店 (僅限非遊戲進行時)。
*   **`Esc`**：返回主選單。

### 商店介面
*   **滑鼠點擊**：購買升級項目。
*   **`B`**：返回主選單。

## 檔案結構
*   `嘉桐gemini.py`: 遊戲主程式。
*   `assets/`: 存放圖片資源的資料夾 (若無圖片，遊戲會自動繪製幾何圖形作為替代)。
